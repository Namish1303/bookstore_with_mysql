First open the SQL folder
using library.sql, create a MySQL database in your computer

As you have created the databse in your computer , you will have to change the (user,password and any other ) requirement in
bookstore/views/views.py  Line 21 - 26

Open the bookstore director and install the required libraries (using requirements.txt)
By now Django should be able to connect to the DBMS

using 'python manage.py runser', start the server and go to the url address.
From there you can move to next pages using the options on each page.
